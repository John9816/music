# 快速启动指南

## 在 Android Studio 中打开项目

### 步骤 1: 打开 Android Studio
启动 Android Studio 应用程序

### 步骤 2: 导入项目
1. 点击 "Open" 或 "Open an Existing Project"
2. 浏览到项目目录: `D:\Projects\music`
3. 点击 "OK"

### 步骤 3: 等待 Gradle 同步
- 首次打开会自动下载依赖包
- 等待底部状态栏显示 "Gradle sync finished"
- 这个过程可能需要 5-10 分钟（取决于网络速度）

### 步骤 4: 配置运行设备
选择以下任一方式：

**方式 A: 使用真机**
1. 用 USB 连接 Android 手机
2. 在手机上开启"开发者选项"和"USB 调试"
3. 在 Android Studio 顶部工具栏选择你的设备

**方式 B: 使用模拟器**
1. 点击工具栏的 "Device Manager"
2. 点击 "Create Device"
3. 选择设备型号（推荐 Pixel 系列）
4. 选择系统镜像（推荐 API 30 或更高）
5. 完成创建并启动模拟器

### 步骤 5: 运行应用
1. 点击工具栏的绿色运行按钮 ▶️
2. 或按快捷键 `Shift + F10`
3. 等待应用安装并启动

## 使用应用

### 首次启动
- 应用会自动加载每日推荐歌曲
- 需要网络连接

### 搜索歌曲
1. 在顶部搜索框输入歌曲名或歌手名
2. 点击"搜索"按钮
3. 浏览搜索结果

### 播放音乐
1. 点击列表中的任意歌曲
2. 底部播放器会显示歌曲信息
3. 点击"播放/暂停"按钮控制播放

## 常见问题

### Q: Gradle 同步失败
**A:** 检查网络连接，可能需要配置代理或使用国内镜像

### Q: 找不到设备
**A:** 确保手机已开启 USB 调试，或重新创建模拟器

### Q: 应用闪退
**A:** 检查 Logcat 日志，可能是网络权限或 API 访问问题

### Q: 无法播放音乐
**A:** 确认网络连接正常，音乐 API 可能暂时不可用

## 项目文件说明

- `build.gradle` - 项目配置文件
- `app/build.gradle` - 应用配置文件
- `app/src/main/AndroidManifest.xml` - 应用清单
- `app/src/main/java/` - Kotlin 源代码
- `app/src/main/res/` - 资源文件（布局、图片、字符串等）
